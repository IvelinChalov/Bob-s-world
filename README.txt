               Bob's world

What is it?
------------

Bob's world is a 2D physics simulator, sand box. Where the user can create simple contraptions similar to the
popular game "Crazy machines".

Installation
------------

The game does not require installation. The user just have to copy the content
of the folder "Bob" on his PC and use the "Bob demo.exe" to start the game.

Used libraries
---------------

Box2d - Used for all the physics in the game. You can download the library from 
http://box2d.org/downloads/.

rapidjson - Used for the serialization/deserialization of the levels in the game. 
You can download it from https://github.com/miloyip/rapidjson.
